# E-Commerce Platform

## 🛒 Project Description

E-Commerce Platform is a fully functional e-commerce web application,
built using HTML, CSS, and JavaScript. 
It includes features like 
product browsing,
shopping cart management,
checkout functionality,
order tracking,
responsive UI,
and dynamic product display.

## 💻 Installation

1. **Clone the repository**

```bash
git clone https://github.com/sakshibhongle1221/AmazonClone.git
```

2. **Navigate to the project folder**

```bash
cd AmazonClone
```

3. **Install dependencies**

```bash
npm install
```

## ▶️ Running the Project :

**Option 1: Open in Browser**
  * Simply open index.html in your web browser:

```bash
open index.html
```

**Option 2: Use Live Server**
  * If you have VS Code Live Server extension, right-click on index.html and select "Open with Live Server"

**Option 3: Live Deployment**
  * Visit the live application: https://clone-17f9d.web.app/

## 🧪 How to Use
* Browse products on the home page (index.html)
* Click on products to view details or add to cart
* Navigate to checkout page to review your items
* Proceed to place an order
* Visit the orders page to track your purchases
* Use the tracking page to get real-time delivery updates
